OPERATORS IN C++

Operators C++ me special symbols hote hain jo variables aur values par different operations perform karte hain. Inki help se hum addition, subtraction, comparison, logical operations aur assignment jaise kaam kar sakte hain. Operators programming ko simple aur efficient banate hain.

Operators are special symbols that perform operations on variables and values. They are used to perform arithmetic, comparison, logical, assignment, and many other operations in C++.

What are Operators?

Operator ek special symbol hota hai jo operands (variables ya values) par operation perform karta hai aur ek result return karta hai.

An operator is a special symbol that performs an operation on one or more operands and produces a result.

💡 Real Life Example

Agar tum calculator me 10 + 5 likhte ho to + ek operator hai aur 10 aur 5 operands hain.

Why Do We Use Operators?

Operators ka use calculations karne, values compare karne, conditions check karne aur variables ki value update karne ke liye kiya jata hai.

Operators are used to perform calculations, compare values, evaluate conditions, and update variables.

  • Perform mathematical calculations.
  • Compare two values.
  • Check logical conditions.
  • Assign values to variables.
  • Increase or decrease values.

Types of Operators

C++ me operators ko unke kaam ke according different categories me divide kiya gaya hai.

C++ operators are classified into different categories based on the operations they perform.

Operator Type Purpose
Arithmetic Performs mathematical operations.
Relational Compares two values.
Logical Checks logical conditions.
Assignment Assigns values to variables.
Increment / Decrement Increases or decreases values.
Bitwise Performs operations on bits.
Ternary Works as a short form of if-else.

Arithmetic Operators

Arithmetic Operators ka use mathematical calculations jaise addition, subtraction, multiplication, division aur remainder nikalne ke liye kiya jata hai.

Arithmetic operators are used to perform mathematical calculations such as addition, subtraction, multiplication, division, and modulus.

Operator Name Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
% Modulus a % b

Arithmetic Operator Example

Neeche diye gaye program me hum sabhi Arithmetic Operators ka use kar rahe hain.

The following program demonstrates the use of all arithmetic operators. 

#include <iostream>

using namespace std;

int main()
{

    int a = 20;
    int b = 5;

    cout << "Addition = " << a + b << endl;

    cout << "Subtraction = " << a - b << endl;

    cout << "Multiplication = " << a * b << endl;

    cout << "Division = " << a / b << endl;

    cout << "Modulus = " << a % b;

    return 0;

}
View output
Output

Addition = 25
Subtraction = 15
Multiplication = 100
Division = 4
Modulus = 0

Relational Operators

Relational Operators do values ko compare karte hain aur result me true ya false return karte hain.

Relational operators compare two values and return either true or false.

Operator Meaning Example
== Equal to a == b
!= Not Equal to a != b
> Greater than a > b
< Less than a < b
>= Greater than or Equal to a >= b
<= Less than or Equal to a <= b

Relational Operator Example

Ye example dikhata hai ki Relational Operators comparison kaise karte hain.

This example shows how relational operators compare two values. 

#include <iostream>

using namespace std;

int main()
{

    int a = 20;
    int b = 10;

    cout << (a > b) << endl;

    cout << (a == b) << endl;

    cout << (a != b);

    return 0;

}
View output
Output

1
0
1

Logical Operators

Logical Operators ka use ek ya ek se zyada conditions ko combine karne ke liye kiya jata hai. Ye mostly if, while aur loops me use hote hain.

Logical operators are used to combine two or more conditions. They return either true or false.

Operator Name Example
&& Logical AND a > 10 && b < 20
|| Logical OR a > 10 || b < 20
! Logical NOT !(a > b)

Logical Operator Example

Is example me AND, OR aur NOT operators ka use dikhaya gaya hai.

This example demonstrates the use of logical operators. 

#include <iostream>

using namespace std;

int main()
{
    int age = 20;

    cout << (age >= 18 && age <= 60) << endl;

    cout << (age < 18 || age > 60) << endl;

    cout << !(age == 20);

    return 0;
}
View output
Output

1
0
0

Assignment Operators

Assignment Operators ka use variables ko value assign karne ke liye kiya jata hai. Kuch assignment operators calculation bhi karte hain aur result ko variable me store kar dete hain.

Assignment operators are used to assign values to variables. Some operators perform calculations and assign the result at the same time.

Operator Meaning Example
= Assign a = 10
+= Add and Assign a += 5
-= Subtract and Assign a -= 2
*= Multiply and Assign a *= 3
/= Divide and Assign a /= 2

Assignment Operator Example

Neeche diya gaya program Assignment Operators ka use dikhata hai.

The following program demonstrates assignment operators. 

#include <iostream>

using namespace std;

int main()
{

int a = 10;

a += 5;

cout << a << endl;

a *= 2;

cout << a;

return 0;

}
View Output
Output

15
30

Increment and Decrement Operators

Increment Operator variable ki value ko 1 se increase karta hai, jabki Decrement Operator value ko 1 se decrease karta hai.

Increment increases a variable by one, while decrement decreases a variable by one.

Operator Name Example
++ Increment ++a
-- Decrement --a 
#include <iostream>

using namespace std;

int main()
{

int a = 5;

a++;

cout << a << endl;

a--;

cout << a;

return 0;

}
View Output
Output

6
5

Bitwise Operators

Bitwise Operators binary bits par operation perform karte hain. Ye mostly low-level programming aur memory manipulation me use hote hain.

Bitwise operators perform operations on binary bits. They are mainly used in system programming and memory manipulation.

Operator Name Example
& Bitwise AND a & b
| Bitwise OR a | b
^ Bitwise XOR a ^ b
~ Bitwise NOT ~a
<< Left Shift a << 2
>> Right Shift a >> 2

Ternary Operator

Ternary Operator if...else ka short form hota hai. Ye condition check karta hai aur uske according result return karta hai.

The ternary operator is a shorthand form of the if...else statement. 

condition ? expression1 : expression2;
View Output 
#include <iostream>

using namespace std;

int main()
{

int age = 20;

(age >= 18) ? cout << "Eligible" : cout << "Not Eligible";

return 0;

}
Output

Eligible

Operator Precedence

Jab ek expression me ek se zyada operators hote hain, tab C++ pehle higher precedence wale operator ko execute karta hai.

Operator precedence decides which operator is evaluated first in an expression. 

int result = 5 + 3 * 2;
Output

11

Important Note

Parentheses () ka use karne se operator precedence change ki ja sakti hai aur code ko zyada readable banaya ja sakta hai.

Using parentheses improves readability and changes the order of evaluation when required.
