LOOPS IN C++

Loops C++ me kisi statement ya statements ke group ko baar-baar execute karne ke liye use kiye jaate hain. Agar hume same task multiple times perform karna ho, to har baar code likhne ki jagah hum loop ka use karte hain. Isse code chhota, readable aur efficient ban jata hai.

Loops are used to execute a block of code repeatedly until a specified condition becomes false. They reduce code repetition and make programs more efficient.

What are Loops?

Loop ek programming structure hai jo kisi block of code ko tab tak repeat karta hai jab tak di hui condition true rehti hai.

A loop is a programming structure that repeatedly executes a block of code as long as the given condition is true.

💡 Real Life Example

Socho teacher ne bola ki apna naam 100 baar likho. Agar loop na ho to tumhe 100 baar same statement likhni padegi. Loop ki help se sirf ek baar code likhkar wahi kaam 100 baar kar sakte ho.

Why Do We Use Loops?

Loops ka use repeated tasks ko automatically perform karne ke liye kiya jata hai. Ye code ko short, fast aur easy banate hain.

Loops are used to perform repetitive tasks automatically. They reduce code duplication and improve efficiency.

  • Avoid writing the same code multiple times.
  • Reduce program length.
  • Improve code readability.
  • Save time and effort.
  • Useful for processing arrays, patterns, and tables.

Types of Loops

C++ me mainly teen types ke loops use kiye jaate hain.

C++ mainly provides three types of loops.

Loop Use
for Loop When the number of iterations is known.
while Loop When the condition is checked before execution.
do...while Loop When the loop should execute at least once.

for Loop

for loop tab use kiya jata hai jab hume pehle se pata ho ki loop kitni baar chalega.

The for loop is used when the number of iterations is known in advance. 

for(initialization; condition; increment/decrement)
{
    // code
}
💡 Best Use

Print tables, numbers, patterns aur fixed number of repetitions ke liye for loop sabse best choice hai.

for Loop Example

Neeche diya gaya program 1 se 5 tak numbers print karta hai.

The following program prints numbers from 1 to 5 using a for loop. 

#include <iostream>

using namespace std;

int main()
{

    for(int i = 1; i <= 5; i++)
    {
        cout << i << " ";
    }

    return 0;

}
View output
Output

1 2 3 4 5

while Loop

while loop tab use kiya jata hai jab hume pehle se nahi pata hota ki loop kitni baar chalega. Ye har iteration se pehle condition check karta hai.

The while loop is used when the number of iterations is unknown. It checks the condition before executing the loop body. 

while(condition)
{
    // code
}
💡 Best Use

Menu-driven programs, user input validation aur condition-based repetition ke liye while loop ka use kiya jata hai.

while Loop Example

Is example me while loop ki help se 1 se 5 tak numbers print kiye gaye hain.

This example prints numbers from 1 to 5 using a while loop. 

#include <iostream>

using namespace std;

int main()
{

    int i = 1;

    while(i <= 5)
    {
        cout << i << " ";
        i++;
    }

    return 0;

}
View output
Output

1 2 3 4 5

do...while Loop

do...while loop me code pehle execute hota hai aur uske baad condition check hoti hai. Isliye ye loop kam se kam ek baar zaroor chalta hai.

The do...while loop executes the code first and checks the condition afterward. Therefore, it always runs at least once. 

do
{
    // code
}
while(condition);
💡 Best Use

Jab program ko minimum ek baar execute karna zaroori ho, tab do...while loop use kiya jata hai.

do...while Loop Example

Is example me do...while loop ki help se 1 se 5 tak numbers print kiye gaye hain.

This example prints numbers from 1 to 5 using a do...while loop. 

#include <iostream>

using namespace std;

int main()
{

    int i = 1;

    do
    {
        cout << i << " ";
        i++;
    }
    while(i <= 5);

    return 0;

}
View output
Output

1 2 3 4 5

Nested Loop

Jab ek loop ke andar dusra loop use kiya jata hai, use Nested Loop kehte hain. Ye patterns, matrices aur tables banane ke liye use hota hai.

A nested loop is a loop inside another loop. It is commonly used for patterns, matrices, and tables. 

for(int i=1; i<=3; i++)
{
    for(int j=1; j<=3; j++)
    {
        cout << "* ";
    }

    cout << endl;
}
💡 Best Use

Nested loops are useful for printing patterns, working with 2D arrays, and creating multiplication tables.

View output
Output

* * *
* * *
* * *

Infinite Loop

Jab loop ki condition kabhi false nahi hoti, tab loop continuously chalta rehta hai. Isse Infinite Loop kehte hain.

An infinite loop runs forever because its condition never becomes false. 

while(true)
{
    cout << "Hello";
}
⚠ Important

Infinite loops generally avoid karne chahiye. Inhe sirf special situations jaise game loops ya servers me use kiya jata hai.

break Statement

break statement loop ko turant stop kar deta hai aur control loop ke bahar chala jata hai.

The break statement immediately terminates the loop. 

#include <iostream>

using namespace std;

int main()
{

    for(int i=1; i<=10; i++)
    {
        if(i==6)
        {
            break;
        }

        cout << i << " ";
    }

    return 0;

}
View output
Output

1 2 3 4 5

continue Statement

continue statement current iteration ko skip karta hai aur next iteration start kar deta hai.

The continue statement skips the current iteration and continues with the next iteration. 

#include <iostream>

using namespace std;

int main()
{

    for(int i=1; i<=5; i++)
    {
        if(i==3)
        {
            continue;
        }

        cout << i << " ";
    }

    return 0;

}
View output
Output

1 2 4 5

Common Mistakes

Beginners loops use karte waqt kuch common mistakes kar dete hain. In mistakes ki wajah se program galat output de sakta hai ya infinite loop me fas sakta hai.

Beginners often make common mistakes while using loops, which may lead to incorrect output or infinite loops.

  • Forgetting to update the loop variable.
  • Using the wrong loop condition.
  • Missing semicolon after do...while.
  • Writing an incorrect initialization value.
  • Creating an unintended infinite loop.

Important Note

Har loop me initialization, condition aur increment/decrement ka sahi use bahut important hai. Agar inme se koi bhi galat ho to program expected result nahi dega.

Every loop should have proper initialization, condition, and increment/decrement. Otherwise, the program may produce incorrect results or run infinitely.

Did You Know?

Loops ka use sirf numbers print karne ke liye nahi hota. Inka use games, searching, sorting, arrays, matrices, file handling aur Artificial Intelligence applications me bhi hota hai.

Loops are widely used in games, searching, sorting, arrays, file handling, graphics, and Artificial Intelligence applications.
