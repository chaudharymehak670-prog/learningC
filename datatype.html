DATA TYPES IN C++

Data Types C++ me variables ke type ko define karte hain. Ye batate hain ki variable kis type ka data store karega, jaise integer, decimal number, character ya text. Data types ki help se compiler memory allocate karta hai aur data ko efficiently manage karta hai.

Data types define the type of data that a variable can store. They help the compiler allocate memory efficiently and perform operations correctly.

What are Data Types?

Data Type ek category hoti hai jo batati hai ki variable kis tarah ki value store karega. Har variable ko declare karte time uska data type likhna zaroori hota hai.

A data type specifies the type of value that a variable can store. Every variable must have a data type before it is used.

Why Do We Use Data Types?

Data Types ka use compiler ko ye batane ke liye kiya jata hai ki variable kitni memory use karega aur us par kis type ke operations perform kiye ja sakte hain.

Data types help the compiler allocate memory, store data efficiently, and perform valid operations on variables.

  • Allocate memory efficiently.
  • Store different types of data.
  • Improve program performance.
  • Reduce programming errors.
  • Make programs easy to understand.

Types of Data Types

C++ me Data Types ko mainly teen categories me divide kiya jata hai.

C++ data types are mainly classified into three categories.

Category Description
Primitive Data Types Basic built-in data types.
Derived Data Types Created from primitive data types.
User-defined Data Types Created by the programmer.

Primitive Data Types

Primitive Data Types C++ ke basic built-in data types hote hain. Inka use numbers, characters, decimal values aur true/false values ko store karne ke liye kiya jata hai.

Primitive data types are the basic built-in data types provided by C++. They are used to store numbers, characters, decimal values, and boolean values.

Data Type Description Example
int Stores whole numbers 25
float Stores decimal numbers 12.5
double Stores large decimal values 25.67895
char Stores a single character 'A'
bool Stores true or false true

Derived Data Types

Derived Data Types basic data types se milkar bante hain. Ye complex data ko store aur manage karne ke liye use hote hain.

Derived data types are created from primitive data types and are used to store more complex data.

  • Array
  • Pointer
  • Function
  • Reference

User-defined Data Types

User-defined Data Types programmer khud create karta hai according to program requirements.

User-defined data types are created by programmers to meet specific program requirements.

  • Class
  • Structure (struct)
  • Union
  • Enumeration (enum)

sizeof() Operator

sizeof() operator kisi bhi data type ya variable ki memory size (bytes me) batata hai.

The sizeof() operator returns the memory size of a data type or variable in bytes. 

cout << sizeof(int);

Data Type Syntax

dataType variableName;

int age;

float salary;

char grade;

Did You Know?

Har Data Type ki memory size compiler aur operating system ke according thodi different ho sakti hai. Isliye memory size check karne ke liye sizeof() operator ka use kiya jata hai.

The size of a data type may vary depending on the compiler and operating system. The sizeof() operator is used to determine the memory size.
