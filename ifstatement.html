CONDITIONAL STATEMENTS IN C++

Conditional Statements C++ me decision lene ke liye use kiye jaate hain. Ye kisi condition ko check karte hain aur uske according different blocks of code execute karte hain. Agar condition true ho to ek block execute hota hai aur agar false ho to doosra block execute hota hai.

Conditional Statements are used to make decisions in a program. They execute different blocks of code depending on whether a condition is true or false.

What are Conditional Statements?

Conditional Statements program ko decision lene ki power dete hain. Ye condition ko evaluate karke decide karte hain ki kaunsa code execute hoga.

Conditional Statements allow a program to make decisions by checking conditions and executing the appropriate block of code.

💡 Real Life Example

Agar traffic signal Green hai to gaadi chalao.

Agar signal Red hai to ruk jao.

Isi tarah C++ bhi condition check karke decision leta hai.

Why Do We Use Conditional Statements?

Conditional Statements ka use alag-alag situations ke hisaab se different actions perform karne ke liye kiya jata hai.

Conditional Statements help programs make decisions and execute different actions based on conditions.

  • Make decisions in programs.
  • Execute different blocks of code.
  • Validate user input.
  • Control program flow.
  • Handle multiple situations efficiently.

Types of Conditional Statements

C++ me mainly paanch types ke Conditional Statements hote hain.

C++ provides five main types of Conditional Statements.

Statement Purpose
if Executes code only when the condition is true.
if...else Chooses one block from two options.
else if Checks multiple conditions.
Nested if Uses one if statement inside another.
switch Selects one option from multiple choices.

if Statement

if statement tab use hota hai jab hume sirf condition true hone par code execute karna ho.

The if statement executes a block of code only when the specified condition is true. 

if(condition)
{
    // code
}
💡 Best Use

Voting eligibility, login validation, marks checking aur age verification jaise tasks me if statement ka use hota hai.

if Statement Example

Neeche diya gaya program check karta hai ki person vote dene ke liye eligible hai ya nahi.

The following program checks whether a person is eligible to vote.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int age = 20;

    if(age >= 18)
    {
        cout << "Eligible for Voting";
    }

    return 0;

}
Output

Eligible for Voting

Explanation
Age is 20.
Condition (age >= 18) is true.
Therefore, "Eligible for Voting" is displayed.

if...else Statement

Jab hume condition ke true aur false dono cases ko handle karna ho tab if...else statement use hoti hai.

The if...else statement executes one block if the condition is true and another block if it is false. 

if(condition)
{
    // code
}
else
{
    // code
}
💡 Best Use

Pass/Fail check karna, Even/Odd check karna aur Login Success/Failure jaise cases me if...else use hota hai.

if...else Example

Ye program check karta hai ki number even hai ya odd.

This program checks whether a number is even or odd.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int num = 8;

    if(num % 2 == 0)
    {
        cout << "Even Number";
    }
    else
    {
        cout << "Odd Number";
    }

    return 0;

}
Output

Even Number

Explanation
The remainder of 8 ÷ 2 is 0.
Therefore, the number is Even.

else if Statement

Jab hume ek se zyada conditions check karni hoti hain tab else if statement use ki jati hai.

The else if statement is used to check multiple conditions one after another. 

if(condition1)
{
    // code
}
else if(condition2)
{
    // code
}
else
{
    // code
}
💡 Best Use

Grades, salary range, discounts aur multiple menu options ke liye else if statement best choice hai.

else if Example

Ye program student ka grade display karta hai.

This program displays the student's grade based on marks.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int marks = 82;

    if(marks >= 90)
    {
        cout << "Grade A";
    }
    else if(marks >= 75)
    {
        cout << "Grade B";
    }
    else if(marks >= 50)
    {
        cout << "Grade C";
    }
    else
    {
        cout << "Fail";
    }

    return 0;

}
Output

Grade B

Explanation
Marks are 82.
82 is greater than or equal to 75 but less than 90.
Therefore, Grade B is displayed.

Nested if Statement

Jab ek if statement ke andar doosra if statement use kiya jata hai, use Nested if kehte hain.

A Nested if statement is an if statement placed inside another if statement. It is used when multiple conditions need to be checked one after another. 

if(condition1)
{
    if(condition2)
    {
        // code
    }
}
💡 Best Use

Nested if ka use tab hota hai jab ek condition true hone ke baad hi doosri condition check karni ho.

Nested if Example

Ye program check karta hai ki student pass bhi hai aur first division me bhi hai ya nahi.

This program checks whether a student has passed and secured the first division.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int marks = 82;

    if(marks >= 33)
    {
        if(marks >= 60)
        {
            cout << "First Division";
        }
    }

    return 0;

}
Output

First Division

Explanation
Marks are 82.
First condition (marks >= 33) is true.
Second condition (marks >= 60) is also true.
Therefore, "First Division" is displayed.

switch Statement

switch statement ka use tab kiya jata hai jab ek variable ki value ke according multiple options me se ek option choose karna ho.

The switch statement is used to select one block of code from multiple options based on the value of a variable. 

switch(expression)
{
    case value1:
        // code
        break;

    case value2:
        // code
        break;

    default:
        // code
}
💡 Best Use

Calculator, menu-driven programs aur day selection jaise applications me switch statement ka use hota hai.

switch Statement Example

Ye program week ke day ko display karta hai.

This program displays the name of the day using the switch statement.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int day = 2;

    switch(day)
    {
        case 1:
            cout << "Monday";
            break;

        case 2:
            cout << "Tuesday";
            break;

        case 3:
            cout << "Wednesday";
            break;

        default:
            cout << "Invalid Day";
    }

    return 0;

}
Output

Tuesday

Explanation
The value of day is 2.
Case 2 matches the value.
Therefore, "Tuesday" is displayed.

default Statement

default block tab execute hota hai jab koi bhi case match nahi karta.

The default statement executes when no case matches the given expression.

break Statement

break statement switch ke current case ko terminate karta hai aur control ko switch block ke bahar le jata hai.

The break statement terminates the current case and exits the switch statement.

Common Mistakes

Beginners Conditional Statements use karte waqt kuch common mistakes kar dete hain. In mistakes ki wajah se program galat output de sakta hai.

Beginners often make common mistakes while using conditional statements, which may produce incorrect results.

  • Using = instead of == in conditions.
  • Missing curly braces { } for multiple statements.
  • Forgetting the break statement in switch cases.
  • Writing incorrect logical conditions.
  • Using unnecessary nested if statements.

Important Note

Hamesha comparison ke liye == aur value assign karne ke liye = operator ka use karein. Ye beginners ki sabse common mistake hoti hai.

Always use == for comparison and = for assignment. This is one of the most common mistakes made by beginners.

Key Points

  • Conditional statements help a program make decisions.
  • if executes code only when the condition is true.
  • if...else handles two possible outcomes.
  • else if is useful for checking multiple conditions.
  • Nested if is used when one condition depends on another.
  • switch is a better choice for multiple fixed options.
  • Always use break inside switch cases.
  • Use == for comparison and = for assignment.
