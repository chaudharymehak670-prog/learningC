FUNCTIONS IN C++

Function C++ me ek reusable block of code hota hai jo kisi specific task ko perform karta hai. Ek baar function bana lene ke baad use program me jitni baar chahe call kiya ja sakta hai. Isse code repeat nahi hota aur program zyada organized aur easy to maintain ban jata hai.

A function is a reusable block of code that performs a specific task. Once a function is created, it can be called multiple times, making programs shorter, organized, and easier to maintain.

What are Functions?

Function ek block of code hota hai jo ek particular task perform karta hai. Jab bhi us task ki zarurat hoti hai, function ko call kar diya jata hai.

A function is a block of code designed to perform a specific task. It runs only when it is called.

💡 Real Life Example

Socho tumhare ghar me washing machine hai.

Jab bhi kapde dhone hote hain, tum sirf machine ka Start button dabate ho.

Har baar machine ko andar se dobara banana nahi padta.

Bilkul isi tarah Function ek baar banta hai aur baar-baar use kiya ja sakta hai.

Why Do We Use Functions?

Functions ka use code ko chhota, reusable aur easy banane ke liye kiya jata hai.

Functions are used to reduce code repetition, improve readability, and make programs easier to manage.

  • Avoid writing the same code repeatedly.
  • Make programs modular.
  • Improve code readability.
  • Simplify debugging.
  • Save development time.

Parts of a Function

Ek Function mainly chaar parts se milkar banta hai.

A function mainly consists of four parts.

Part Description
Function Declaration Declares the function.
Function Definition Contains the function body.
Function Call Executes the function.
Return Statement Returns a value to the caller.

Function Syntax

Har Function ka ek fixed syntax hota hai.

Every function follows a fixed syntax in C++. 

returnType functionName()
{
    // code
}
💡 Syntax Explanation
  • returnType → Function kis type ki value return karega.
  • functionName() → Function ka naam.
  • { } → Function body jahan code likha jata hai.
  • return → Function se value wapas bhejne ke liye.

Function Declaration

Function Declaration compiler ko function ke naam, return type aur parameters ke baare me batata hai. Isme sirf function ka prototype likha jata hai.

A function declaration tells the compiler about the function's name, return type, and parameters. It is also called a function prototype. 

void greet();
💡 Best Use

Function Declaration tab useful hota hai jab function ki definition baad me likhi jati hai.

Function Definition

Function Definition ke andar actual code likha jata hai jo function call hone par execute hota hai.

A function definition contains the actual code that is executed when the function is called. 

void greet()
{
    cout << "Welcome";
}
💡 Best Use

Function Definition me wahi logic likha jata hai jo baar-baar use karna ho.

Function Call

Function ko execute karne ke liye uska naam likhkar call kiya jata hai.

A function call executes the function whenever it is needed in the program. 

greet();
💡 Best Use

Ek hi function ko program me multiple places par call kiya ja sakta hai.

Return Statement

return statement function se value ko wapas caller ke paas bhejta hai.

The return statement sends a value back to the calling function. 

int square()
{
    return 25;
}
💡 Best Use

Jab function koi result calculate karke wapas dena ho tab return use kiya jata hai.

Function Example

Neeche diya gaya program ek simple function create aur call karta hai.

The following program creates and calls a simple function.

View Example 
#include <iostream>

using namespace std;

void greet()
{
    cout << "Welcome to C++";
}

int main()
{
    greet();

    return 0;
}
Output

Welcome to C++

Explanation
The function greet() is created.
Inside main(), the function is called.
When the function executes, it prints "Welcome to C++".

Types of Functions

C++ me Functions ko Parameters aur Return Value ke basis par mainly chaar types me divide kiya jata hai.

Functions in C++ are mainly classified into four types based on parameters and return values.

Type Description
No Parameters, No Return Value Does not take input and does not return anything.
Parameters, No Return Value Takes input but does not return a value.
No Parameters, Return Value Returns a value without taking input.
Parameters, Return Value Takes input and returns a value.

Function Without Parameters

Is type ke function me koi input pass nahi kiya jata.

A function without parameters does not receive any input from the user. 

void message()
{
    cout << "Hello";
}
💡 Best Use

Jab har baar same task perform karna ho, tab parameter ki zarurat nahi hoti.

Function With Parameters

Is type ke function me values arguments ke through pass ki jati hain.

A function with parameters receives input values from the calling function. 

void add(int a, int b)
{
    cout << a + b;
}
💡 Best Use

Different values ke saath same function ko baar-baar use karne ke liye parameters use kiye jaate hain.

Function With Parameters Example

Ye program do numbers ka sum print karta hai.

This program adds two numbers using a function with parameters.

View Example 
#include <iostream>

using namespace std;

void add(int a, int b)
{
    cout << "Sum = " << a + b;
}

int main()
{

    add(10,20);

    return 0;

}
Output

Sum = 30

Explanation
The values 10 and 20 are passed to the function.
The function adds both numbers.
Finally, the result is displayed.

Function With Return Value

Ye function calculation karke result ko return karta hai.

A function with a return value performs a task and sends the result back to the calling function. 

int square(int num)
{
    return num * num;
}

Function With Return Value Example

Ye program kisi number ka square return karta hai.

This program returns the square of a number.

View Example 
#include <iostream>

using namespace std;

int square(int num)
{
    return num * num;
}

int main()
{

    cout << square(6);

    return 0;

}
Output

36

Explanation
The value 6 is passed to the function.
The function calculates 6 × 6.
The result is returned and displayed.

Inline Function

inline function ek chhota function hota hai jise compiler function call ki jagah directly insert kar deta hai. Isse execution thoda fast ho sakta hai.

An inline function is expanded by the compiler at the place where it is called, reducing the overhead of function calls. 

inline int square(int n)
{
    return n * n;
}
💡 Best Use

Inline functions are best for small functions that are called frequently.

Recursive Function

Jab koi function khud ko hi call karta hai, use Recursive Function kehte hain.

A recursive function is a function that calls itself until a stopping condition is reached. 

int factorial(int n)
{
    if(n==1)
        return 1;

    return n * factorial(n-1);
}
💡 Best Use

Recursion is useful for solving problems like factorial, Fibonacci series, tree traversal, and divide-and-conquer algorithms.

Recursive Function Example

Ye program recursion ka use karke factorial calculate karta hai.

This program calculates the factorial of a number using recursion.

View Example 
#include <iostream>

using namespace std;

int factorial(int n)
{
    if(n==1)
        return 1;

    return n * factorial(n-1);
}

int main()
{

    cout << factorial(5);

    return 0;

}
Output

120

Explanation
The function calls itself repeatedly.
5 × 4 × 3 × 2 × 1 = 120
Finally, the result is returned and displayed.

Common Mistakes

  • Forgetting to call the function after defining it.
  • Using the wrong return type.
  • Not returning a value from a non-void function.
  • Passing the wrong number or type of arguments.
  • Writing recursive functions without a base condition.

Important Note

Har function ka ek clear purpose hona chahiye. Chhote aur reusable functions program ko readable aur maintainable banate hain.

Each function should perform one specific task. Small and reusable functions improve code quality and readability.

Did You Know?

Modern software applications me hundreds ya thousands of functions hote hain. Functions hi bade programs ko manageable banate hain.

Large software projects are built using hundreds of reusable functions, making development easier and more organized.
