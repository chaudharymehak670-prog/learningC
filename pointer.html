POINTERS IN C++

Pointer C++ ka ek special variable hota hai jo kisi dusre variable ki value nahi, balki uska memory address store karta hai. Pointers ka use memory management, arrays, functions aur dynamic memory allocation me kiya jata hai. Ye C++ ke sabse powerful concepts me se ek hai.

A pointer is a special variable that stores the memory address of another variable instead of storing its actual value. Pointers are widely used in memory management, arrays, functions, and dynamic memory allocation.

What is a Pointer?

Pointer ek aisa variable hota hai jo kisi dusre variable ka memory address store karta hai.

A pointer is a variable that stores the memory address of another variable.

💡 Real Life Example

Socho tumhare dost ka ghar hai.

Ghar ke andar rehna ek alag baat hai, lekin ghar ka address alag hota hai.

Pointer ghar ke andar ki value nahi, balki us ghar ka address store karta hai.

Why Do We Use Pointers?

Pointers ka use memory ko efficiently access karne aur bade programs ko fast aur flexible banane ke liye kiya jata hai.

Pointers help access memory efficiently and are useful for dynamic memory allocation, arrays, functions, and data structures.

  • Store memory addresses.
  • Access variables indirectly.
  • Pass data efficiently to functions.
  • Support dynamic memory allocation.
  • Used in linked lists, trees, and other data structures.

Pointer Declaration

Pointer declare karne ke liye data type ke baad * symbol use kiya jata hai.

A pointer is declared using the * symbol after the data type. 

int *ptr;
💡 Syntax Explanation
  • int → Data type.
  • * → Indicates that the variable is a pointer.
  • ptr → Pointer variable name.

Address Operator (&)

& operator kisi variable ka memory address return karta hai.

The & operator returns the memory address of a variable. 

int num = 10;

💡 Best Use

Address operator ka use pointer ko kisi variable ka address dene ke liye hota hai.

Pointer Initialization

Pointer ko use karne se pehle usme kisi variable ka address store karna padta hai. Is process ko Pointer Initialization kehte hain.

Pointer initialization is the process of assigning the address of a variable to a pointer. 

int num = 10;

int *ptr = #
💡 Best Use

Pointer ko initialize karne ke baad hi uske through variable ko access kiya ja sakta hai.

Dereference Operator (*)

* operator ka use pointer ke andar stored address par maujood value ko access karne ke liye kiya jata hai.

The * operator is called the dereference operator. It is used to access the value stored at the memory address held by a pointer. 

int num = 10;

int *ptr = #

cout << *ptr;
💡 Best Use

Dereference operator ka use pointer ke through original variable ki value ko read ya modify karne ke liye hota hai.

Ye program ek variable ki value aur uska memory address pointer ki help se display karta hai.

This program displays the value and memory address of a variable using a pointer.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int num = 50;

    int *ptr = #

    cout << "Value = " << *ptr << endl;

    cout << "Address = " << ptr;

    return 0;

}
Output

Value = 50
Address = 0x61ff08

Explanation
The variable num stores the value 50.
The pointer ptr stores the address of num.
Using *ptr, we access the value stored at that address.
The memory address may be different on every computer.

Changing Value Using Pointer

Pointer ki help se original variable ki value ko bhi change kiya ja sakta hai.

A pointer can also be used to modify the value of the original variable. 

int num = 20;

int *ptr = #

*ptr = 100;

cout << num;
💡 Best Use

Pointers allow direct modification of variables without accessing them by name.

Changing Value Example

Ye program pointer ki help se variable ki value change karta hai.

This program changes the value of a variable using a pointer.

View Example 
#include <iostream>

    int num = 20;

Output

Explanation
The pointer stores the address of num.
Using *ptr = 100, the value stored at that address is changed.

NULL Pointer

Agar pointer kisi bhi valid memory address ko point nahi karta, to use NULL Pointer kehte hain.

A NULL pointer is a pointer that does not point to any valid memory location. 

int *ptr = NULL;
💡 Best Use

NULL Pointer ka use ye batane ke liye hota hai ki pointer abhi kisi memory address ko point nahi kar raha.

Pointer Arithmetic

Pointer Arithmetic ka use pointer ko next ya previous memory location par move karne ke liye kiya jata hai.

Pointer arithmetic allows a pointer to move to the next or previous memory location. 

int arr[3] = {10,20,30};

int *ptr = arr;

ptr++;

cout << *ptr;
💡 Best Use

Pointer Arithmetic arrays ko efficiently traverse karne ke liye use hoti hai.

Pointer Arithmetic Example

Ye program pointer ko next memory location par move karta hai.

This program moves the pointer to the next element of an array.

View Example 
#include <iostream>

using namespace std;

int main()
{

    int arr[] = {10,20,30};

    int *ptr = arr;

    ptr++;

    cout << *ptr;

    return 0;

}
Output

20

Explanation
Initially, the pointer points to the first element (10).
After ptr++, it moves to the second element.
Therefore, *ptr prints 20.

Pointer to Pointer

Jab ek pointer dusre pointer ka address store karta hai, use Pointer to Pointer kehte hain.

A pointer to pointer is a pointer that stores the address of another pointer. 

int num = 10;

int *ptr = #

int **pptr = &ptr;
💡 Best Use

Pointer to Pointer advanced programming aur dynamic memory management me use hota hai.

Ye program Pointer to Pointer ka use dikhata hai.

This program demonstrates the use of a pointer to pointer.

View Example 
#include <iostream>

using namespace std;

{

    int num = 10;

Void Pointer

Void Pointer kisi bhi data type ka address store kar sakta hai, lekin use directly dereference nahi kiya ja sakta.

A void pointer can store the address of any data type, but it must be typecast before dereferencing. 

int num = 10;

void *ptr = #
💡 Best Use

Void Pointer generic programming aur memory management me use hota hai.

Wild Pointer

Agar pointer declare to kar diya jaye lekin usme koi valid memory address assign na kiya jaye, to use Wild Pointer kehte hain.

A wild pointer is a pointer that has been declared but not initialized with a valid memory address. 

int *ptr;
💡 Best Use

Wild Pointer ka use avoid karna chahiye kyunki ye unpredictable behavior aur program crash ka reason ban sakta hai.

Dangling Pointer

Dangling Pointer wo pointer hota hai jo aisi memory ko point karta hai jo free ya delete ho chuki ho.

A dangling pointer points to a memory location that has already been deleted or released. 

int *ptr = new int(10);

delete ptr;
💡 Best Use

Memory delete karne ke baad pointer ko NULL kar dena ek achhi practice hai.

Common Mistakes

Beginners pointers use karte waqt kuch common mistakes karte hain jo errors ya crashes ka reason ban sakti hain.

Beginners often make common mistakes while using pointers, which may lead to errors or program crashes.

  • Using an uninitialized pointer.
  • Dereferencing a NULL pointer.
  • Accessing memory after deleting it.
  • Forgetting to initialize a pointer.
  • Confusing * and & operators.

Important Note

Pointer ko use karne se pehle hamesha initialize karein. Agar pointer kisi valid address ko point nahi karta, to uska use program ko crash kar sakta hai.

Always initialize pointers before using them. Never dereference an invalid or NULL pointer.

Did You Know?

Pointers are one of the most powerful features of C++. They are widely used in operating systems, game engines, database systems, and embedded programming.
