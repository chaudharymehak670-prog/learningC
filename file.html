FILE HANDLING IN C++

File Handling C++ ka ek important concept hai jiska use files me data ko store, read aur update karne ke liye kiya jata hai. Isse data permanently save rehta hai aur program band hone ke baad bhi delete nahi hota. C++ me File Handling ke liye <fstream> header file ka use kiya jata hai.

File Handling in C++ is used to create, read, write, and update files. It helps store data permanently so that it is available even after the program is closed.

What is File Handling?

File Handling ek process hai jisme hum files ke andar data ko store aur retrieve karte hain.

File Handling is the process of creating, reading, writing, and managing files in a program.

💡 Real Life Example

Socho tum ek diary me roz apne notes likhte ho.

Diary band karne ke baad bhi tumhare notes safe rehte hain.

Isi tarah File Handling data ko permanently save karti hai.

Why Do We Use File Handling?

File Handling ka use data ko permanently store karne aur baad me use dobara access karne ke liye kiya jata hai.

File Handling is used to permanently store data, retrieve it later, and manage information efficiently.

  • Store data permanently.
  • Read saved information.
  • Update existing files.
  • Share data between programs.
  • Maintain records efficiently.

Main Concepts of File Handling

File Handling me mainly ye concepts use hote hain.

The following concepts are commonly used in File Handling.

Concept Description
fstream Header file for file operations.
ofstream Used to write data into a file.
ifstream Used to read data from a file.
fstream Used for both reading and writing.

Basic Syntax

File Handling shuru karne ke liye sabse pehle <fstream> header file include ki jati hai.

The <fstream> header file is required to perform file operations in C++. 

#include <fstream>
💡 Syntax Explanation
  • <fstream> provides classes for file operations.
  • ofstream writes data into a file.
  • ifstream reads data from a file.
  • fstream can perform both reading and writing.

ofstream (Output File Stream)

ofstream ka use file ke andar data likhne (write) ke liye kiya jata hai.

The ofstream class is used to create a file and write data into it. 

ofstream file("student.txt");

file << "Hello World";

file.close();
💡 Best Use

Student records, reports aur log files save karne ke liye ofstream use hota hai.

ifstream (Input File Stream)

ifstream ka use file se data read karne ke liye kiya jata hai.

The ifstream class is used to read data from a file. 

ifstream file("student.txt");

string text;

file >> text;

file.close();
💡 Best Use

Already saved information ko read karne ke liye ifstream use kiya jata hai.

fstream (File Stream)

fstream ka use ek hi object se file ko read aur write dono karne ke liye kiya jata hai.

The fstream class can perform both reading and writing operations on a file. 

fstream file("student.txt");
💡 Best Use

Jab ek hi file me read aur write dono operations karne ho tab fstream use hota hai.

Opening a File

File par koi bhi operation karne se pehle usse open karna zaroori hota hai.

A file must be opened before performing read or write operations. 

ofstream file;

file.open("student.txt");
💡 Best Use

File ko open() function ki help se manually open kiya ja sakta hai.

Writing Data into a File

Ye program ek file create karta hai aur usme data write karta hai.

This program creates a file and writes data into it.

View Example 
#include <iostream>
#include <fstream>

using namespace std;

int main()
{

    ofstream file("student.txt");

    file << "Welcome to C++ File Handling";

    file.close();

    cout << "Data Written Successfully";

    return 0;

}
Output

Data Written Successfully

Explanation
The file student.txt is created.
The text "Welcome to C++ File Handling" is written into the file.
Finally, the file is closed using close().

Appending Data to a File

Agar existing file ke end me naya data add karna ho to ios::app mode use kiya jata hai.

The ios::app mode is used to add new data at the end of an existing file without deleting its previous content. 

ofstream file("student.txt", ios::app);

file << "New Record";
💡 Best Use

Attendance, logs aur student records me naya data add karne ke liye ios::app use hota hai.

Appending Data Example

Ye program existing file ke end me naya data add karta hai.

This program appends new data to an existing file.

View Example 
#include <iostream>
#include <fstream>

using namespace std;

int main()
{

    ofstream file("student.txt", ios::app);

    file << "\nWelcome Again";

    file.close();

    cout << "Data Appended Successfully";

    return 0;

}
Output

Data Appended Successfully

Explanation
The file is opened in append mode.
The new text is added at the end of the existing file.
The previous data remains unchanged.
Finally, the file is closed.

Common Mistakes

Beginners File Handling use karte waqt kuch common mistakes kar dete hain.

Beginners often make common mistakes while working with files.

  • Forgetting to include the <fstream> header file.
  • Not checking whether the file opened successfully.
  • Forgetting to close the file.
  • Using the wrong file mode.
  • Trying to read from a file that does not exist.

Important Note

File ka kaam complete hone ke baad hamesha close() function call karein. Ye ek achhi programming practice hai.

Always close a file after completing file operations to save data properly and free system resources.
