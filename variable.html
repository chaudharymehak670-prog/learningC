VARIABLES IN C++

Variables computer ki memory me data store karne ke liye use hote hain. Har variable ka ek unique naam, data type aur value hoti hai. Program ke execution ke dauran variable ki value ko change kiya ja sakta hai. Variables ki help se hum user input store kar sakte hain, calculations perform kar sakte hain aur data ko easily manage kar sakte hain.

Variables are used to store data in the computer's memory. Every variable has a unique name, a data type, and a value. Variables help store, update, and manage data during program execution.

What are Variables?

Variable ek named memory location hota hai jisme data store kiya jata hai. Jab bhi program ko kisi value ko future me use karna hota hai, us value ko variable me store kiya jata hai.

A variable is a named memory location used to store data. It allows programmers to store and use values whenever required in a program.

Why Do We Use Variables?

Variables ka use data ko store karne, update karne aur program ke different parts me reuse karne ke liye kiya jata hai. Ye program ko flexible aur easy banate hain.

Variables are used to store, modify, and reuse data. They make programs flexible, readable, and easy to manage.

  • Store data in memory.
  • Reuse values multiple times.
  • Perform calculations.
  • Take user input.
  • Improve program readability.

Rules for Naming Variables

Variable name rakhte waqt kuch important rules follow karne padte hain.

Variable names must follow certain rules in C++.

  • Variable name must start with a letter or underscore (_).
  • It cannot start with a number.
  • Spaces are not allowed.
  • Special characters are not allowed except underscore (_).
  • C++ keywords cannot be used as variable names.
  • Variable names are case-sensitive.

Variable Declaration

Variable Declaration ka matlab variable ka naam aur uska data type batana hota hai. Declaration ke time variable ko value dena zaroori nahi hota.

Variable declaration means specifying the data type and name of a variable. A value is not required during declaration. 

dataType variableName;

Variable Initialization

Variable Initialization ka matlab variable ko value assign karna hota hai.

Variable initialization means assigning a value to a variable. 

int age;

age = 20;

Declaration and Initialization

Variable ko declare aur initialize ek hi statement me bhi kiya ja sakta hai.

A variable can be declared and initialized in the same statement. 

int age = 20;

Multiple Variable Declaration

Ek hi data type ke multiple variables ko ek hi statement me declare kiya ja sakta hai.

Multiple variables of the same data type can be declared in a single statement. 

int a, b, c;

Constant Variables

Agar kisi variable ki value ko change nahi karna ho to const keyword ka use kiya jata hai.

The const keyword is used to create variables whose values cannot be changed. 

const float PI = 3.14;

Variable Syntax

dataType variableName;

dataType variableName = value;
View Examples 
#include <iostream>

using namespace std;

int main(){

int age = 21;

string name = "Anjali";

cout << age;

cout << name;

return 0;

}
Output
21
Anjali
